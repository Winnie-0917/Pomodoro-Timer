Lo‑Fi Focus Space
==================

極簡番茄鐘 (Pomodoro) 範例頁面，包含：

- 25 分工作 / 5 分休息 預設
- 白噪音背景（用 Web Audio API 生成）與上傳音檔功能
- 動態漸層背景與圓形進度顯示

使用方法
-------

1. 在本機打開 `index.html`（直接在瀏覽器中開啟）。
2. 使用「開始」啟動計時器；使用「暫停」或「重設」。
3. 在背景聲下拉選單可以選擇「白噪音」或「上傳音檔」。
4. 用音量滑桿調整背景音量。

備註
----

- 本專案僅使用純前端技術（HTML/CSS/JS），不需後端。
- 上傳的音檔只在本地播放（不會上傳到伺服器）。
